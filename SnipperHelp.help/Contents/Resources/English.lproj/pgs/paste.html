Paste a Snippet

There are two ways to paste a snippet. Either using the mouse or using the global hotkeys.

Paste using the mouse:

  1. Click onto the title field of the main Snipper window and hold down the mouse button until a transparent picture of the clicked snippet shows up.

  2. Drag the picture onto the text field you want the Snippet to be pasted. Keep the mouse button pressed.

  3. Release the mouse button to paste the Snippet at the current position.


Paste using the global hotkeys:

  1. Place the text cursor on the position you want the Snippet to be entered.

  2. Simultaneously press the Snipper hotkeys (?) and the number of the Snippet you want to insert (1-9).
